النظام الآن جاهز تقنياً بنسبة 100% من حيث الأكواد، ولكن لكي يعمل فعلياً على جهازك، هناك 4 خطوات يدوية يجب عليك القيام بها الآن:

### 1. تثبيت المكتبات (في الـ Terminal الخاص بك)
تأكد من أنك داخل المجلد الرئيسي للمشروع، ثم قم بتثبيت كافة المتطلبات لضمان عدم ظهور خطأ "Module Not Found":

```
pip install -r requirements.txt
```
### 2. تجهيز الموديلات المحلية (Ollama)
إذا كنت ستستخدم الخيار المحلي (وهو الافتراضي)، يجب أن تتأكد من تحميل الموديلات داخل برنامج Ollama:

- افتح الـ Terminal وشغل الأوامر التالية:
  ```
  ollama pull nomic-embed-text
  ollama pull llama3
  ``` ملاحظة: تأكد أن برنامج Ollama يعمل في شريط المهام (Background).
### 3. إكمال بيانات ملف الـ .env
افتح الملف وقم بتعبئة البيانات الحقيقية:

- OPENAI_API_KEY : ضع مفتاحك إذا كنت تريد استخدام OpenAI.
- DRUG_API_URL : الرابط الحالي هو openFDA للتجربة، إذا كان لديك API خاص بك، استبدله هنا.
- DRUG_API_KEY : ضع مفتاح الـ API الخاص ببياناتك إذا كان يتطلب ذلك.
### 4. التشغيل الأول (بناء قاعدة البيانات)
لأنك لم تبنِ قاعدة البيانات بعد، يجب تشغيل النظام مع أمر "إعادة البناء" لمرة واحدة فقط:

```
python app.py --rebuild
```
سيقوم النظام بجلب البيانات من الـ API، توليد الـ Embeddings، وحفظها في ملف faiss_index.bin .
